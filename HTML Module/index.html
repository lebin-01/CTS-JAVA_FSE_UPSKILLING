Debugging with Chrome DevTools

Tips for inspecting and debugging this portal live in the browser.

Inspect Element

Right-click any element → Inspect. Modify CSS live in the Styles panel — changes reflect instantly without reload.

Console Tab

Open DevTools → Console. All console.log() calls from this page appear here. Try typing localStorage to inspect saved preferences.

Breakpoints

Go to Sources tab → click a line number in the JS to set a breakpoint. Reload to pause execution and inspect variable values.

Network Tab

Track resource loading, video streaming, and any API calls. Filter by type: XHR, Media, Doc.

Application Tab

View and edit localStorage and sessionStorage directly here — useful for testing the Preferences feature above.

Device Toolbar

Press Ctrl+Shift+M to simulate mobile viewports and test the responsive layout of this portal.
